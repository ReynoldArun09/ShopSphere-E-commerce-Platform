# ShopSphere-E-commerce-Platform

## Tech Stack

### Frontend

- **React 19** — Component-based UI architecture with modern concurrent features
- **TypeScript** — Strong typing and improved developer tooling across the entire codebase
- **Tailwind CSS** — Utility-first styling for rapid and consistent UI development
- **Vite / Modern React Tooling** — Lightning-fast development environment and optimized builds

### Backend Services

- **Node.js** — Primary runtime for backend microservices
- **TypeScript** — Strongly typed backend services for maintainability
- **Microservice Architecture** — Independent for modular domain separation
- **Redis** — High-speed caching layer for improving performance and reducing database load
- **Kafka** — Event streaming and asynchronous messaging for scalable communication between services
- **Razorpay** — Secure and seamless payment gateway integration

### Storage & Media

- **Cloudinary** — Cloud media storage and real-time image optimization

### DevOps & Deployment

- **Docker & Docker Compose** — Containerized environment for consistent local and production setups
